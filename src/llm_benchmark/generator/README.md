# Generator

This module provides utilities for generating test data.

## Modules

### gen_list.py

Test data generation:
- `GenList.random_list(n, m)` - Generate a list of n random integers from 0 to m
- `GenList.random_matrix(n, m)` - Generate an nÃ—n matrix of random integers from 0 to m

These generators are used to create input data for benchmark tests.
