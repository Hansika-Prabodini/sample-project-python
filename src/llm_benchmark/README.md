# LLM Benchmark

Core benchmarking module for the llm-benchmarking-py project.

## Overview

This module contains a collection of algorithm and data structure implementations designed for benchmarking. Each submodule focuses on specific problem domains.

## Submodules

- **algorithms/** - Prime number and sorting algorithms
- **control/** - Control flow and nested loop operations
- **datastructures/** - List and basic data structure operations
- **generator/** - Test data generation utilities
- **sql/** - SQL query execution and benchmarking
- **strings/** - String manipulation operations
