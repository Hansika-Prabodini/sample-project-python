# Data Structures

This module contains implementations of basic data structure operations.

## Modules

### dslist.py

List manipulation operations:
- `DsList.modify_list(v)` - Add 1 to each element in a list
- `DsList.search_list(v, n)` - Find all indices where a value appears
- `DsList.sort_list(v)` - Sort a list and return a copy
- `DsList.reverse_list(v)` - Reverse a list
- `DsList.rotate_list(v, n)` - Rotate list by n positions
- `DsList.merge_lists(v1, v2)` - Merge two lists

These operations benchmark basic list processing and manipulation patterns.
